# Alpha Implementation Notes

Alpha documentation is for developer-only implementation notes and unstable designs.

Use alpha pages for:

- internal design sketches,
- early architecture notes,
- implementation details that may change before beta,
- questions that need developer validation.

Alpha pages should still be concrete. Do not use them as dumping grounds for placeholders.
